# Helping-Hands-NGO-s
Helping Hands NGO Website is a web-based platform designed to help NGOs manage donations and volunteers efficiently.

Key Features:
• View information about the NGO and its social activities.
• QR code–based donation system for easy and quick payments.
• Volunteer registration system to allow users to join the NGO.
• Admin login system to securely access the admin dashboard.
• Admin dashboard to manage volunteer records and view donation details.
• Data stored securely in a database for better management and transparency.
• Simple and user-friendly interface for donors, volunteers, and administrators.

Methodologies:
– The project follows Modular Programming, where different modules such as Home Page, Donation System,
  Volunteer Registration, and Admin Panel are developed separately for better organization and maintenance.
– The system uses Structured Programming principles to maintain clean, readable, and well-organized code.
– Form validation is implemented to ensure correct user input such as name, email, phone number and
  donation details.
– The project follows an Admin-controlled architecture, where only the admin can access and manage
  the dashboard and records.
  
Technologies:
– Python as the main programming language.
– Flask framework for backend development and routing.
– HTML for creating the structure of web pages.
– CSS for designing and styling the website interface.
– SQLite Database for storing volunteer and donation data securely.

Tools:
– IDE: VS Code for coding and development.
– Web Browser: Google Chrome for testing the website.
– Operating System: Windows and Android for development and testing.
